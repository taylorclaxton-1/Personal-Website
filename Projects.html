Building a SQL Server Database - Operation School Bell

ERDs, Logical & Physical Design, what's more to love?! Last semester I was tasked with working with a wonderful group of people in my database course to design a fully functioning SQL Server Database from the ground up, with each design element having a strict time constraint! The organization that our database was based upon worked with local school systems to provide clothing, books, toys, and other necessities to students from elementary through high school. This database fulfilled the need for them to track their inventory, purchases, volunteer, student, and school data. The database provided an easy way to run reports that answered critical questions to the operation.
